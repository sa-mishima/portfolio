三点リーダー症候群チェッカー
https://santen-leader.nankakaku.com/

コンセプト 自分のツイートにどのくらい三点リーダーが含まれているかを調べられるサービス
公開日 2021年2月2日
使用ツール Visual Studio Code
使用言語（担当） HTML, CSS, jQuery
制作メンバー 2名（デザイナー、プログラマー各1名）
制作期間 1日半
制作背景 三点リーダーをつい使ってしまうという、『三点リーダー症候群』がTwitterのトレンドに上がった。
実際に自分のツイートでどのくらい使っているのか確かめてみたくなったので、チェッカーを作った。
ポイント
  • ・トップページに実際の解析結果のスクリーンショットを掲載することで、ユーザーに使用イメージを持ってもらえるようにした。
  • ・Twitterのトレンドワードは入れ替わりが激しいので、スピード感を優先して制作した。そのためにデザインツールは使用せず、コーディングをした後に最低限のデザインを適用させた。
